2、constructor和instanceof？

constructor是JavaScript中对象的属性，保存着该对象的构造函数。

instanceof是二元操作符，用于检查一个对象是否是某个构造函数的实例。

3、js数据类型都有哪些？

七中原始数据类型：string字符串类型；number数字类型；boolean布尔类型；null表示空值；undefined未定义的值；symbol独一无二的值；

两种复杂数据类型：object队形类型，表示由多个属性和方法组成的复杂类型；function函数类型，表示可以被执行的代码块；

6、讲一下js中的包装类型？

基础类型的数据在使用时候，js引擎会先将之包装为对象，语句执行完对象被销毁。这个过程也被称为“装箱拆箱”。

7、讲一下js类型的隐式转换？

JavaScript 中的类型隐式转换指的是在执行运算时，JavaScript 引擎会自动将操作数转换为相应的数据类型，以便执行运算。例如，如果一个操作数是字符串类型，而另一个操作数是数字类型，JavaScript 引擎会自动将字符串转换为数字，然后再执行运算。

13、如何判断一个对象为空？

使用 Object.keys() 方法检查对象是否包含任何属性。如果对象没有任何属性，那么它就是空的。

使用 for...in 循环来遍历对象的属性，如果没有任何属性可以遍历，那么对象就是空的。

使用 JSON.stringify() 方法将对象转换为 JSON 字符串，然后检查字符串是否为 “{}”。如果是，那么对象就是空的。

15、number类型最大值是多少？如果后台发的数据超过这个值怎么办？

大多数浏览器最大值是2^53，最小是-2^53。最大值和最小值可以通过Number.MAX_VALUE和Number.MIN_VALUE查看。

后台发送数据超过这个值可以用字符串类型代替。

16、12和12.0有什么区别？

没有区别，因为整型值存储空间是浮点类型的一半，所以js会把12.0转为12存储。
